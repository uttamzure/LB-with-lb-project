Summary (Classic Load Balancer) :-

The Classic Load Balancer (CLB) is one of AWSs legacy load balancing services. It operates at both Layer 4 (TCP) and Layer 7 (HTTP/HTTPS), distributing incoming traffic across multiple EC2 instances to improve application availability and reliability. CLB supports basic features such as SSL termination and sticky sessions, making it suitable for small to medium-scale applications. However, for modern and advanced workloads, AWS recommends using Application Load Balancer (ALB) or Network Load Balancer (NLB).
